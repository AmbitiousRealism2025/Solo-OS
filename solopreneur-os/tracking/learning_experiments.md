## R&D vs Procrastination Check

Use this test for any learning activity:

| Signal | R&D (Keep) | Procrastination (Kill) |
|--------|------------|------------------------|
| Has deadline? | Yes | No |
| Has test? | Yes | No ("just exploring") |
| Will ship something? | Yes | No |
| $1,000 investment test? | Would invest | Wouldn't invest |
| After 2 weeks, tangible progress? | Yes | No |

**If 3+ answers are "Procrastination" â†’ Kill it.**
